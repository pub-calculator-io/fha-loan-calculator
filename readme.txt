The FHA loan calculator lets you plug in the numbers and calculate the monthly payments, interest, and total amount paid over time.

[https://www.calculator.io/fha-loan-calculator/](https://www.calculator.io/fha-loan-calculator/)

== Usage ==

Add the FHA Loan Calculator shortcode to your page, post or sidebar:

`[ci_fha_loan_calculator]`

Add the following code to your template where you would like the FHA Loan Calculator to appear:

`<?php display_ci_fha_loan_calculator(); ?>`

== Installation ==

1. Upload the FHA Loan Calculator /ci_fha_loan_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the FHA Loan Calculator plugin through the "Plugins" menu in WordPress.
